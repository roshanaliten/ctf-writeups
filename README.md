# CTF Write-ups

This repository contains my write-ups for Capture The Flag (CTF) challenges from platforms such as TryHackMe, Hack The Box, and others.

## Overview

The purpose of this repository is to document my practical learning in cybersecurity and penetration testing. Each write-up follows a structured approach covering the steps taken to solve the challenge.

## Contents

- Reconnaissance
- Service enumeration
- Exploitation
- Privilege escalation
- Key takeaways

## Platforms

- Hack The Box
- PortSwigger

## Structure

Write-ups are organized by platform. Each folder contains individual challenge reports with step-by-step explanations and commands used.
